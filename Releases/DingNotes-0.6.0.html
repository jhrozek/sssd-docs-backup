Highlights

libini_config

  • libini now supports validators that check for well-formed INI files. Please see the Doxygen documentation for more details on using the new functions. The new functions include:
    • ini_rules_read_from_file
    • ini_rules_check
    • ini_rules_destroy
    • ini_errobj_*

Note for distribution packagers

  • API and ABI is backward compatible with last release (0.5.0)
