Highlights

  • Fix for older openldap libraries that can fail setting the "no canonicalization" option
  • pam_sss will now return authinfo_unavail when the SSSD daemon is shut down, instead of PAM_SYSTEM_ERR
  • Fixes a regression in LDAP failover handling
  • Adds support for using Glib to provide unicode support instead of libunistring
    • SSSD must be built with the configure flag --with-unicode-lib=glib2
  • All user/group/netgroup names passed to NSS/PAM are now validated to be valid UTF-8 strings
