Highlights

  • Addresses CVE-2010-4341 - DoS in sssd PAM responder can prevent logins
  • Vast performance improvements when enumerate = true
  • All PAM actions will now perform a forced initgroups lookup instead of just a user information lookup
    • This guarantees that all group information is available to other providers, such as the simple provider.
  • For backwards-compatibility, DNS lookups will also fall back to trying the SSSD domain name as a DNS discovery domain.
  • Support for more password expiration policies in LDAP
    • 389 Directory Server
    • FreeIPA
    • ActiveDirectory
  • Support for ldap_tls_{cert,key,cipher_suite} config options
    • Provided by community member Tyson Whitehead
  • Assorted bugfixes
