Highlights¶

  • Removed the Data Provider process. All Data Provider backends will now speak directly to the responders.
  • Remove the MagicPrivateGroups option. This will be an internal feature of backends that support it.
  • Add first steps towards a native FreeIPA data provider.
  • Support password changing in LDAP and Kerberos providers.
  • Added Python API for managing SSSD configuration.
