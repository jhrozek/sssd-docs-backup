SSSD Tools

SSSD Takes advantage of several common tools that can be found in the source tree.

  • Collection - a set of interfaces to manage data in sets of key-value pairs (stable).
  • INI - interface to read ini files and retrieve values from them (stable)
  • DHASH - API that implements a hash table object (stable).
  • Path Utils - API to manage directory and file paths (stable).

All these tools now have their own ​git repository, but they still use the SSSD Trac instance for tracking issues.
